Automating Translation

Production Tool

After Effects (ScriptUI) · DeepL API

A recurring bottleneck in global campaign delivery was localization. The same assets going out in multiple languages, each requiring a motion designer to manually re-enter translated text, check styling, and re-export. The process added days to every international rollout and introduced consistent formatting errors.

I built an After Effects ScriptUI panel that lives directly alongside the composition. Select a text layer, choose a target language, and the script calls the DeepL API to fetch the translation, duplicates the layer, and populates it with the translated copy.

A localization step that used to consume most of a day now takes minutes, with formatting preserved throughout.
