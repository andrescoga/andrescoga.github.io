Lyric Sync Pipeline

Production Tool

Python · After Effects (JSX)

For the 2025 holiday campaign I produced over 15 lyric videos across multiple artists.

I built a pipeline that eliminates that step. It takes as inputs the official lyrics and the song file and runs a forced-alignment algorithm developed to generate a word-level timed SRT file. A Python script converts that into a JSX file, which After Effects imports automatically, creating all text layers with their exact timing already set and picking up any existing text formatting.

What used to take hours of manual scrubbing per video now takes under 5 minutes. Timing is handled, everything creative stays with the motion designer. I use it on every lyric video I produce.
