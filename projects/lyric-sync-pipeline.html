Lyric Sync Pipeline

Production Tool

For the Christmas campaign of 2025 I was tasked to create over 15 lyric videos. One of the most tedious parts of the process is syncing the lyrics to the song — which arguably takes the most time. Frustrated by spending all this time on setup instead of focusing on the creative side, I developed a tool that automates the most labor-intensive step entirely. The interface asks only for the official lyrics and the song, and through a forced-alignment process developed by the Institut de Recherche et Coordination Acoustique/Musique it returns a timed SRT file with the raw data of the timing for each word. From there, a Python script converts that SRT into a JSX file, which is automatically imported into After Effects and generates all the text layers with their exact in and out points already set, adopting any previous formatting automatically. What would have taken hours of manual scrubbing now takes minutes. But more importantly, the pipeline stops there by design. The timing is handled; everything else remains in the hands of the motion designer. I know use this tool for all the lyric videos i produce, saving time and making the creating process much more flexible and enjoyable.
