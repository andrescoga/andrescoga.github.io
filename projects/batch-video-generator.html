Batch Video Generator

Production Tool

For every album release, social platforms require pseudo-videos — a looping visual, typically the album cover animation delivered to DSPs in square format, trimmed to the length of each individual track. For a 12-song album, that means 12 exports, each manually trimmed, rendered, and renamed. It's straightforward work, but at album scale it becomes a significant time drain on producers and motion designers.

I built a Python tool using ffmpeg that reduces this to a single step. The program prompts for the base animation file and a tracklist with song durations, then automatically generates one video per track — each looping the animation and trimmed to the exact length of that song — named and ready for publishing. The interface is simple enough for producers and marketers to run it themselves without any technical setup.

What previously took a designer the better part of a day now runs unattended in a few minutes, freeing the team to focus on work that actually requires creative judgment.
