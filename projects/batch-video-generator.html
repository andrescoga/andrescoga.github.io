Batch Video Generator

Production Tool

Python · ffmpeg

I built a Python tool using ffmpeg that generates pseudo-videos for DSPs — the album cover animation looped and trimmed to the length of each track. It takes the base animation file and a tracklist with durations, then outputs one video per track, looping, trimming to exact length, and naming correctly for publishing. The interface is simple enough for producers and marketers to run without technical setup.

For an album, what usually takes hours of manual export work now runs unattended in under 3 minutes.
