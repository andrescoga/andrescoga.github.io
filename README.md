# andrescoga.github.io

Personal portfolio website for Andrés Escobar — video editor and motion designer based in New York City.

**Live site:** [andrescoga.com](https://andrescoga.com)

## Features

- **Interactive video previews** — Hover to preview projects on desktop, tap to expand on mobile
- **Motion design gallery** — Lazy-loaded video snippets with hover-to-play
- **Responsive design** — Optimized experience across desktop, tablet, and mobile
- **Fast video delivery** — Bunny.net CDN with Vimeo fallback

## Tech Stack

- Vanilla JavaScript (ES6+)
- HTML5 & CSS3
- GitHub Pages hosting
- Bunny.net CDN for video

## Local Development

No build tools required. Open `index.html` in a browser or run a local server:

```bash
# Python
python -m http.server 8000

# Node.js (if http-server is installed)
npx http-server
```

## Deployment

Push to the `main` branch — GitHub Pages deploys automatically.
